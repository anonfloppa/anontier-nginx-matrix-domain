Privacy Policy

When using the anontier.nl matrix server, the default config is used with the following changes:

  • There are triggers added to the postgresql database to prevent synapse from saving the IP address in the database. Link
  • New devices created are named "new anontier device" by default
  • User agent is not saved in the database
  • Redactions and message deletions are purged after one minute
  • Nginx access logs are disabled, and error logs are critical only.
  • Nginx logs are not saved to disk (volatile)
  • IP addresses are not shown in the nginx logs
  • Tor connections will always be allowed
